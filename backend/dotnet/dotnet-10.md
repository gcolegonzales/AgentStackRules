# .NET 10 Version-Specific Rules

## Overview
.NET 10 is the latest release with continued performance and developer experience improvements.

## Key Differences
- Continue using the minimal hosting model.
- Take advantage of the latest C# language features.
- Review release notes for new APIs and deprecations when starting new work.

## Preferred Patterns
- Follow the same conventions as .NET 8 unless new patterns are introduced.
- Use the latest recommended patterns from Microsoft documentation.

## Common Pitfalls
- When using bleeding-edge features, verify that all dependencies and libraries support .NET 10.
- Third-party library compatibility may lag behind the framework release.
